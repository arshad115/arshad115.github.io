# Label plot lines in matplotlib

Use `plt.legend()` to show the labels.

```python
plt.plot(x_A,y_A,'g--', label="plot A")
plt.plot(x_B,y_B,'r-o', label="plot A")
plt.legend()
plt.show()
```
